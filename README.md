# The Pope Raid Tactics

Static one-page raid tactics site for the fictional MMO encounter **The Pope**, with Cardinals and Swiss Guards as adds.

## GitHub Pages setup

1. Make the repository public, or use a GitHub plan/settings that allow Pages for private repositories.
2. Go to **Settings → Pages**.
3. Set **Source** to `Deploy from a branch`.
4. Set branch to `main` and folder to `/root`.
5. Save.

The page should then be available at:

`https://jwrpalmer99.github.io/PopeRaidBoss/`

The site is self-contained in `index.html`; the raid diagram image is embedded directly in the page.
